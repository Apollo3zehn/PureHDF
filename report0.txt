report:
  Generate report into stdout from a previously generated gcdump or from a running process.

Usage:
  dotnet-gcdump report [options] [<gcdump_filename>]

Arguments:
  <gcdump_filename>    The file to read gcdump from.

Options:
  -p, --process-id <process-id>    The process id to collect the gcdump from.
  -t, --report-type <HeapStat>     The type of report to generate. Available options: heapstat (default) [default: HeapStat]
  -?, -h, --help                   Show help and usage information
